Installation Guide

  1. Select Ethereum Network (Mainnet, Holesky, Sepolia)
  2. Select Execution Client (Besu, Geth, Nethermind)
  3. Select Consensus Client (Lighthouse, Nimbus, Prysm, Teku)
  4. Select MEVBoost (On or Off)
  5. Enter Ethereum Address to receive fees / rewards
  6. Import Validator Keystore and Password (Optional)
